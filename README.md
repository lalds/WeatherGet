# Weather & Time Desktop Widget

мой пет проект показывающий погоду и температуру в зависимости от региона/города в котором вы проживаете

## Основные возможности
*   **Часы и дата:** Обновляются каждую секунду.
*   **Погода:** Данные загружаются через OpenWeatherMap API.
*   **Автоопределение города:** При запуске определяет ваше местоположение по IP.
*   **Кастомизация:** Возможность вручную изменить город при запуске.
*   **Интерфейс:** 
    *   Прозрачное окно без рамок.
    *   Режим "Поверх всех окон" (Always on Top).
    *   Возможность свободно перетаскивать виджет по экрану мышкой.
